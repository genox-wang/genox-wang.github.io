Android Studio 安装配置

发表于 2019-02-01 更新于 2021-04-09 分类于 tools

简介

Android Studio 是一个Android开发环境，基于 IntelliJ IDEA. 类似 Eclipse ADT，Android Studio 提供了集成的 Android 开发工具用于开发和调试。
虽然比 VSCode 重，但由于它的调试功能比较优秀。所以在后期调试会发挥作用
